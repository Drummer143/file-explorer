# File Explorer

## Features:
  1. Navigating through your file system
  2. Showing updates in current directory
  3. Creating/renaming/deleting files
  4. Desktop App
  5. Switching languages

## Technology stack
  1. Electron
  2. Typescript
  3. React
  4. Zustang
  5. Webpack
  6. TailwingCSS + Sass
  7. i18n
